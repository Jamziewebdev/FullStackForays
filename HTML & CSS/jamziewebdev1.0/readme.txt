This website was created with the dual purpose of finishing an assignment in a course, and to serve as a portfolio/CV-website. 

The assignment was:

Create a multi-page, responsive and accessible website. It should be written with semantic HTML and should contain at least one:

* Heading
* List
* Image
* Menu (navigation)
* Embedded video
* Downloadable file
* Form

The website should contain one page about the history of the internet, one page about an important person, company or event in the history of the internet, as well as a page about how to implement agile principles in a webdev-project.
